Review Tips

Scheduled Tasks: Look for tasks with unusual names, unknown publishers, or suspicious commands (powershell, cmd, encoded).
Registry Run Keys: These run at every startup. Verify all entries come from trusted, signed software.
Startup Folders: Files here run at login. Scripts (.vbs, .bat, .ps1) are higher risk than signed shortcuts.
Services: Unsigned services outside Program Files/System32 deserve investigation. Check binary signature and path.
WMI Persistence: Legitimate WMI subscriptions are rare. Any finding here warrants close examination.
Defender Exclusions: Malware commonly adds itself to exclusions. Review and remove any you don't recognize.
Browser Extensions: Extensions with broad permissions (all URLs, native messaging) can see all your browsing data.
Event Logs: Event 7045 = new service installed. Events 4625/4740 = failed logins/lockouts.
Hosts File: Entries redirecting microsoft.com, google.com, or banking sites are major red flags.
Network: Established connections to unknown IPs, especially on unusual ports, may indicate C2 communication.
Alternate Data Streams: Hidden data attached to normal files. Legitimate uses exist but can hide malicious payloads.
General: Research unknown findings before taking action. False positives happen. Use the checkboxes to track your review progress.
